# tick

This fetches the latest stock price for the given ticker symbol.

To fetch the current price for [HCP](https://finance.yahoo.com/quote/HCP/):

```bash
モ tick HCP
55.55
```

## Installation

```bash
モ go install github.com/xlgmokha/tick@latest
```
